# SOPL 2025 – Interactive Dashboard (Template)

Use this as a starting point. It ships with a small sample CSV.
## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Cloud (via GitHub)
1. Push this folder to a new GitHub repo.
2. In Streamlit Cloud → New app → choose this repo → `app.py` as the main file.
